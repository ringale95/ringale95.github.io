This module largely revolves around the ways code can be executed by providing data in multiple ways. The ways can be categorized as,
  • Command Line Arguments
  • Command Line StdInput
    • Standard Input Keyboard
    • Simulate Keyboard action by < Input Redirection operator to file
  • Hardcode values stored in Variables

Command Line Arguments

One of the ways to pass data to client-code is via command line arguments. These arguments are accessible within your public static void main(String args[]) as args array. You can then reference it using the array syntax to fetch the data stored.

Code: 
public class HelloGoodbye {
    public static void main(String[] args) {
        // printing using command line arguments:
        String arg1 = args[0];
        String arg2 = args[1];

        System.out.println("Hello " + arg1 + " and " + arg2 + ".");
        System.out.println("Goodbye " + arg2 + " and " + arg1 + ".");
    }
}
Command Line: 
> javac HelloWorld.java 
> java HelloWorld Hello World

Standard Input Keyboard

Once any Java code is compiled and run using command line, we can interact with StdInput using system's Keyboard to pass desired data to the code.

Code: 
import edu.princeton.cs.algs4.StdIn;
import edu.princeton.cs.algs4.StdRandom;

public class RandomWord {
    public static void main(String[] s) {

        // StdIn.readString(): reads and returns the next string from standard input.
        // StdIn.isEmpty(): returns true if there are no more strings available on standard input, and false otherwise.
        // StdOut.println(): prints a string and terminating newline to standard output. It’s also fine to use System.out.println() instead.
        // StdRandom.bernoulli(p): returns true with probability p
        // and false with probability 1−p

        String champion = null;
        int count = 0;
        // read words until there are no more words left
        while (!StdIn.isEmpty()) { // just like hasNext() it returns true if there are no more string available on std input else false
            String word = StdIn.readString(); // just like next() it reads and returns next string
            count++;

            if (StdRandom.bernoulli(1.0 / count)) // similar to Math.random() < 1.0 / count
                champion = word;
        }

        if (champion != null) {
            System.out.println(champion);
        }


    }

}
Command Line: 
> javac RandomWord.java 
> java RandomWord
> head tails // Values passed from keyboard
> head // Output printed on command line by code

Simulate Keyboard action by < Input Redirection operator to file

We can give the StdInput using any file by simulating Keyboard action via the '<' Input redirection operator on the command line. This helps us give large amount of data through StdInput.

Code: 
import edu.princeton.cs.algs4.StdIn;
import edu.princeton.cs.algs4.StdOut;
import edu.princeton.cs.algs4.UF;

public class UFDynamicClient {
    public static void main(String[] args) {
        int n = StdIn.readInt();
        UF uf = new UF(n);
        while (!StdIn.isEmpty()) {
            int p = StdIn.readInt();
            int q = StdIn.readInt();
            if (uf.find(p) != uf.find(q)) {
                uf.union(p, q);
                StdOut.println(p + " " + q);
            }
        }
    }
}
Command Line: 
> javac UF.java 
> java UF < tinyTxt.txt // tinyTxt.txt contains the dataset
