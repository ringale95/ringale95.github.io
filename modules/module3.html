Analysis of Algorithm

Terminologies

n
Input size given to the program
T(n)
Running time of the program under analysis
Input model
Data given to program in the form of data-structure

Concepts

Observations
Values obtained by running the algorithm against multiple value of n.
Hypothesis
Supposition or proposed explanation based on limites Observation.
Model
Implements the hypothesis for predictions, e.g., T(n) is the running time as a function of n.
Prediction
Outcome of model for the given input size n

Types of order of growth

Constant
The order of growth is 1. 
a = b + c
Logarithmic
The order of growth is log N. 
Binary search divides in half the search space
Linear
The order of growth is N. 
double max = a[0];
for(int i = 0; i < N; i++)
    if(a[i] > a[max])
        max = a[i];
Linearithmic
The order of growth is NlogN . 
Divide and conquer algorithm - MergeSort
Quadratic
The order of growth is N2. 
for(int i = 0; i < N; i++)
    for(int j = i + 1; j < N; j++)
        if(a[i] + a[j] == 0)
            cnt++;
Cubic
The order of growth is N3. 
for(int i = 0; i < N; i++)
    for(int j = i + 1; j < N; j++)
        for(int k = i + 2; k < N; ;++)
        if(a[i] + a[j] + a[k] == 0)
            cnt++;
Exponential
The order of growth is 2N. 
Exhaustive search
