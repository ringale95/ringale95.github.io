What is an Iterables?

Iterables are interfaces in java.lang package.An object implementing Iterables allows it to be iterated. Three ways in which object can be iterated are:

  • Enhanced for loop
  • Iterable forEach loop
  • Iterator<T> interface

Enhanced for loop

Objects of Classes implementing Collection interface can be iterated using for-each loop, Collection interface extends Iterable interface.

Code: 
public class IterateUsingEnhancedForLoop {
    public static void main(String[] args) {
       List<String> list = new ArrayList<>();
       list.add("Apple");
       list.add("Mango");
       list.add("Banana");

       // iterate through list
       for(String element : list)
        System.out.println(element);
    }
}
Output: 
Apple
Mango
Banana

Iterable ForEach

forEach() method takes Lambda Expression as a parameter. Lambda expression is called for each element of the collection.

Code: 
public class IterateUsingForEach{

    public static void main(String[] args){
        List<String> list = new ArrayList<>();
        list.add("Apple");
        list.add("Banana");
        list.add("Kiwi");

        // iterate using forEach loop
        list.forEach((element) -> 
        {
            System.out.println(element)
        });
    }
}
Output: 
Apple
Mango
Banana

Lambda Expression

Lambda expressions provide a way to write piece of code that can be treated as a method. We can use lambda expression with interfaces which has only 1 Abstract methodcalled as Functional Interfaces

Instead of creating separate classes to implement Calculator, lambda expressions (int a, int b) -> a + b and (int a, int b) -> a - b provide the implementation directly where they are needed

Code: 
public Interface Calculator{
    int calculate(int a, int b);
}

// lambda expression:
Calculate add = ( int a, int b) ->  a + b;
Calculate subtract = ( int a , int b) -> a - b;
Output: 
System.out.println(add.calculate(5,3))      // 8
System.out.println(subtract.calculate(5,3)) // 2

Iterate using Iterable

We can iterate the elements of Iterable by obtaining the Iterator using iterator() method

Methods used are:

  • hasNext()
  • next()
  • remove()

Iterator

Iterator is an interface that provides way to access elements of collection. It is part of java.util package and it is used in Java Collections framework to iterate over elements. Class Example implementing Iterator interface

Code: 
public Interface Iterator<E>{

    // if iteration has more elements returns true;
    boolean hasNext();

    //returns current element and move to next
    E next()
}

Implementation of Iterator interface

A class implementing the Interface Iterator will have all the implementations of abstract methods
