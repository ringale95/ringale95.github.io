What is LinkedList?

LinkedList is a recursive data structure that is either empty(null) or reference to a node having generic item and a reference to a linked list. Benefit is it is easier to insert or remove element than in array

LinkedList are faster than resizing array but it might be slower and take more space than array. If we want to perform and preserve all the data quickly should go with linkedlist. Resizing array will call resize() method when the stack of size is power of 2

  • Node : Contains parameterized Item parameterized type and a Node. It is private class inside main class and not accessible by clients

Insert a node in the beginning

Code: 
public class InsertNodeAtBeginning {
    
    private class Node{
        Item item;
        Node next;
    }

   private Node first;          // first node
   private int N;               // number of items

   public void push(Item item){
    Node oldFirst = first;
    first = new Node();
    first.item = item;
    first.next = oldFirst;
    N++;
   }   
}

Remove a node from beginning

Code: 
public Item RemoveFromBeginning(){
    Item item = first.item;
    first = first.next();
    N--;
    return item
}
